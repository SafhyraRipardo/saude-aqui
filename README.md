# Saúde Aqui - Repositório do Projeto

Bem-vindo ao repositório do projeto "Saúde Aqui", uma plataforma dedicada à divulgação de informações relevantes sobre saúde. Este site simples, desenvolvido em HTML, CSS e JavaScript, oferece uma interface intuitiva para acessar e explorar uma lista de notícias relacionadas à saúde.

## Recursos Principais

- **Design Responsivo:** Desfrute de uma experiência de usuário consistente em dispositivos de diferentes tamanhos, graças ao design responsivo incorporado.
- **Bootstrap:** Utilizamos o framework Bootstrap para garantir uma apresentação visual atraente e uma estrutura sólida.
- **Lista de Notícias Dinâmica:** A seção "Lista de Notícias" é alimentada dinamicamente pelo arquivo `blog.js`, proporcionando uma atualização contínua e fácil manutenção das informações.

## Estrutura do Projeto

- **Pasta `src`:** Contém o arquivo `blog.js`, responsável por fornecer dados dinâmicos para a lista de notícias.
- **Arquivo `style.css`:** Gerencia o estilo e a aparência do site, garantindo uma apresentação visualmente agradável.
